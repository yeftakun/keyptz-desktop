keyPTZ-desktop

Utilitas kecil untuk mengubah input keyboard menjadi Virtual Xbox 360 Controller agar kontrol kamera PTZ di vMix lebih fleksibel.

Download v1.1.1 All Release

Fitur

  • Keyboard mapping ke virtual Xbox 360 controller
  • Dukungan gamepad fisik dan virtual secara bersamaan
  • Modifier key, boost key, dan boost multiplier
  • Hold/Cruise control
  • Editor config dan profile dari tray

Catatan Penggunaan

  • Untuk shortcut kamera PTZ di vMix, set persentase analog dan trigger ke 100%.
  • Jalankan keyPTZ sebelum vMix, dan jangan exit sebelum vMix dimatikan.

Lokasi Data

Data pengguna disimpan di:

%LOCALAPPDATA%\keyPTZ-desktop
